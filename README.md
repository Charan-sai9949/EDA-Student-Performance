# Exploratory Data Analysis (EDA) - Student Performance Dataset

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Student Performance Dataset using Python. The objective is to understand the dataset, identify trends, analyze correlations, and visualize student performance using statistical techniques and charts.

## Objectives

- Analyze the dataset
- Perform statistical summary
- Identify missing values and duplicate records
- Create data visualizations
- Find correlations between variables
- Identify factors affecting final grades

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset Information

- Records: 395
- Features: 33
- Format: CSV

## Visualizations

- Histogram
- Correlation Heatmap
- Scatter Plot
- Box Plot
- Bar Chart
- Pie Chart
- Count Plot
- Line Plot

## Key Findings

- No missing values found.
- No duplicate records found.
- G2 has the strongest positive correlation with G3.
- G1 also strongly influences G3.
- Study time has a slight positive effect on final grades.
- Failures have a negative impact on student performance.
